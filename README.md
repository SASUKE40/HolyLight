# 圣光幻灯片

### 简介

* 圣光幻灯片是一个迷你的歌词放映程序，允许用户进行歌词存储和放映，可以自定义背景，标题内容的字体风格字号及颜色
* 原始安装包大小：19M
* 添加运行库后：72M
* 更新：2015-02-04
* 版本号：1.0.0 Beta
* 平台：Win8.1/Win8/Win7/WinXP(不推荐)
* 运行环境：.NET Framework 4
* Github地址：[HolyLight](https://github.com/SASUKE40/HolyLight)
